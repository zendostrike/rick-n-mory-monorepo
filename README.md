# ReactNMortyMonorepo

Instalar dependencias

```
npm install --legacy-peer-deps
```

## Correr web

```
npm run start:web
```

## Correr mobile

Instalar antes los pods o los paquetes de gradle.

En una consola:

```
npm run start:start
```

Y en otra consola:

```
npm run start:run-ios
```
